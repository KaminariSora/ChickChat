Log in

กด Log in เข้าสู่ระบบ ในการเข้าสู่ระบบครั้งแรก user จะได้รับน่องไก่ 5 น่อง การเข้าสู่ระบบแต่ละวัน user จะได้รับสิทธิ์การส่ง like ให้ 10 ครั้งต่อวัน

Create Avatar

เมื่อทำการ Log in เรียบร้อยแล้ว user จะสามารถสร้าง avatar เพื่อใช้ในการพูดคุย นอกจากนี้ยังสามารถติด Tag บอกความรู้สึกหรือสถานะได้

Random Chat

เมื่อ user ทำการสร้าง avatar และติด Tag เป็นที่เรียบร้อย user ต้องทำการกด Random chat เพื่อเข้าสู่ห้องสนทนา

Like you

ในแต่ละวัน user สามารถส่ง Like ให้คู่สนทนาได้ไม่เกิน 10 ครั้ง โดย Like จะถูกแปรเปลี่ยนเป็นน่องไก่

Avatar shop

user สามารถตกแต่ง avatar ได้ตามความชอบโดยสามารถซื้อไอเท็มตกแต่งได้ที่ Avatar shop โดยจะใช้น่องไก่ในการซื้อ

Chick shop

user สามารถซื้อน่องไก่ได้ที่ Chick shop
